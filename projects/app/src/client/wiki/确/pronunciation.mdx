**🗣️ Pronunciation of 确 (què)**

- **Pinyin:** què
- **Tone: Fourth tone** — sharp **falling** tone, like giving a command: "Yes!"

**🔤 Breakdown:**

- **q** like **"ch"** in "cheese" (but more like **"chy"**)
- **uè** sounds like **"way"** but with fourth tone → sharp drop
- **què** sounds like **"chway!"** with a sharp falling command

**🎧 Tone tip:**

The **fourth tone** (`) is a **sharp falling** tone:

Say it like you're stating something definitively: **"què!"** — that's the tone pattern of **què**.

**📝 Common Examples:**

- 确 (què) - "solid; firm"
- 确定 (què dìng) - "to confirm; certain"
- 确实 (què shí) - "indeed; really"
- 确保 (què bǎo) - "to ensure; to guarantee"

**💡 Memory Tip:**

When you're **certain** about something **solid**, you state it with conviction — that definitive,
falling tone matches perfectly with **què**!
