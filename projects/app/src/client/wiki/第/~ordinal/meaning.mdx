Used before numerals to form ordinal numbers; indicates sequence or ranking.

## Quick Reference

| Aspect         | Info                     |
| -------------- | ------------------------ |
| Pinyin         | dì                       |
| Core meaning   | ordinal marker; sequence |
| Part of speech | prefix, ordinal marker   |
| Tone           | fourth tone (falling)    |

## Visual Breakdown

第 represents **ordering and ranking**:

| Component | Visual Description                                            |
| --------- | ------------------------------------------------------------- |
| **竹**    | Bamboo radical - represents organized, systematic arrangement |
| **弟**    | Younger brother - represents sequential order and hierarchy   |

## Mnemonic

Think of 第 as **organizing like bamboo sections in brotherly order**:

- Bamboo (竹) grows in organized sections, like a natural numbering system
- Younger brother (弟) represents order in family hierarchy
- Like arranging bamboo sticks in sequential order
- Each section/brother has a specific position in the sequence

This creates the concept: **systematic ordering and sequential arrangement**.

## Core Meaning & Usage

第 transforms cardinal numbers into **ordinal numbers (first, second, third, etc.)**. It's used for:

1. **Sequential order**: 第一 (dì yī) - "first"
2. **Rankings**: 第三名 (dì sān míng) - "third place"
3. **Chapters/sections**: 第五课 (dì wǔ kè) - "lesson five"
4. **Time periods**: 第二天 (dì èr tiān) - "the second day"

## Examples

- **第一次** (dì yī cì) - "first time"
- **第二个** (dì èr gè) - "the second one"
- **第十页** (dì shí yè) - "page ten"
- **第一名** (dì yī míng) - "first place"
- **第几** (dì jǐ) - "which number (in sequence)"

## Essential Ordinal Numbers

- **第一** (dì yī) - "first"
- **第二** (dì èr) - "second"
- **第三** (dì sān) - "third"
- **第四** (dì sì) - "fourth"
- **第五** (dì wǔ) - "fifth"

## Grammar Pattern

**第 + Number + (Classifier) + Noun**

- 第一个人 (dì yī gè rén) - "the first person"
- 第三本书 (dì sān běn shū) - "the third book"
- 第五天 (dì wǔ tiān) - "the fifth day"

## Cultural Context

第 is essential for:

- **Academic contexts**: course numbers, test rankings
- **Competition results**: sports rankings, contest positions
- **Sequential events**: describing order of occurrence
- **Formal organization**: systematic arrangement and classification

第 is fundamental for expressing order and sequence in Chinese.
