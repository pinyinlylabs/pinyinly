**🗣️ Pronunciation of 靠 (kào)**

- **Pinyin:** kào
- **Tone: Fourth tone** — sharp **falling** tone, like giving a command

**🔤 Breakdown:**

- **k** like **"k"** in "kind"
- **ào** sounds like **"ow"** in "cow", but with fourth tone → sharp falling
- **kào** sounds like **"cow!"** with a sharp drop

**📝 Common Examples:**

- 靠 (kào) - "lean on; depend on"
- 依靠 (yī kào) - "rely on; depend on"
- 靠近 (kào jìn) - "close to; approach"
- 可靠 (kě kào) - "reliable; dependable"

**💡 Memory Tip:**

The **sharp falling tone** sounds like firmly leaning into something for support — **kào**
