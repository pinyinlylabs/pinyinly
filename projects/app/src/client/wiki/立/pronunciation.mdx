**🗣️ Pronunciation of 立 (lì)**

- **Pinyin:** lì
- **Tone: Fourth tone** — sharp **falling** tone, like giving a command

**🔤 Breakdown:**

- **l** like **"l"** in "light"
- **ì** sounds like **"ee"** in "see", but with fourth tone → sharp drop
- **lì** sounds like **"lee!"** with a sharp falling command

**🎧 Tone tip:**

The **fourth tone** (ˋ) is a **sharp falling** tone:

Start high and drop sharply: **"lì!"** — like giving a firm command or showing determination.

**📝 Common Examples:**

- 立 (lì) - "stand"
- 立刻 (lì kè) - "immediately"
- 建立 (jiàn lì) - "establish"
- 独立 (dú lì) - "independent"
- 站立 (zhàn lì) - "stand up"

**💡 Memory Tip:**

Think of someone commanding **"Lee!"** to stand up — the sharp fourth tone matches the decisive
action of standing!
