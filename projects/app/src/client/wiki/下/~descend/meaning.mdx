To move from a higher to a lower level; below; under; to descend.

## Quick Reference

| Aspect         | Info                           |
| -------------- | ------------------------------ |
| Pinyin         | xià                            |
| Core meaning   | down; below; descend; get off  |
| Part of speech | verb, preposition, directional |
| Tone           | fourth tone (falling)          |

## Visual Breakdown

下 represents **movement downward** in a simple, direct way.

| Component | Visual Description                                          |
| --------- | ----------------------------------------------------------- |
| **一**    | Horizontal line representing a reference level or boundary  |
| **丨**    | Vertical line extending downward below the reference level  |

## Mnemonic

Think of 下 as **pointing downward**:

- The horizontal line (一) is like a table or platform
- The vertical stroke (丨) shows something dropping down below it
- Like an arrow pointing down, or something falling off a table
- The vertical line "descends" below the horizontal reference line

This simple visual directly represents the concept of **going down** or **being below**.

## Core Meaning & Usage

下 represents **downward movement, lower position, or descending action**. It's used:

1. **As "down/below"**: 楼下 (lóuxià) - "downstairs"
2. **Descent action**: 下楼 (xià lóu) - "go downstairs"
3. **Getting off**: 下车 (xià chē) - "get off the bus/car"
4. **Time periods**: 下午 (xiàwǔ) - "afternoon" (after midday)

## Examples

- **下面** (xiàmiàn) - "below; underneath"
- **下来** (xiàlái) - "come down"
- **下去** (xiàqù) - "go down"
- **下雨** (xiàyǔ) - "to rain" (water comes down)
- **下次** (xiàcì) - "next time"
- **放下** (fàngxià) - "put down"

## Directional Usage

下 is frequently used as a directional complement:
- **坐下** (zuòxià) - "sit down"
- **拿下** (náxià) - "take down"
- **写下** (xiěxià) - "write down"

The character emphasizes the completion of downward movement or placement in a lower position.
