**🗣️ Pronunciation of 馆 (guǎn)**

- **Pinyin:** guǎn
- **Tone: Third tone** — **falling-rising** tone, like saying "uh-huh" when nodding thoughtfully

**🔤 Breakdown:**

- **g** like **"g"** in "go"
- **uǎn** sounds like **"wahn"** but with the third tone → dip down and rise up
- **guǎn** sounds like **"gwahn"** with a dip-then-rise pattern

**🎧 Tone tip:**

The **third tone** (ˇ) is a **fall–then-rise** tone. Start middle, dip down low, then rise back up -
like you're thoughtfully considering a place to visit.

**📝 Common Examples:**

- 图书馆 (tú shū guǎn) - "library"
- 博物馆 (bó wù guǎn) - "museum"
- 体育馆 (tǐ yù guǎn) - "gymnasium"
- 饭馆 (fàn guǎn) - "restaurant"
- 旅馆 (lǚ guǎn) - "hotel, inn"
- 茶馆 (chá guǎn) - "teahouse"

**💡 Memory Tip:**

When you're deciding which museum or library to visit and think "Hmm, which one..." - that
contemplative tone matches the dip-and-rise third tone of **guǎn**.
