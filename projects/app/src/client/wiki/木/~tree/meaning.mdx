Tree; wood; timber; wooden; made of wood.

## Quick Reference

| Aspect         | Info                           |
| -------------- | ------------------------------ |
| Pinyin         | mù                             |
| Core meaning   | tree; wood; timber; wooden     |
| Part of speech | noun                           |
| Tone           | fourth tone                    |

## Visual Breakdown

木 is a pictographic representation of a tree.

| Component | Meaning                                   |
| --------- | ----------------------------------------- |
| **木**    | Shows a tree with trunk, branches, and roots |

The character depicts the essential structure of a tree - vertical trunk with horizontal branches.

## Mnemonic

Think of 木 as **"a tree standing tall with branches"**:

- The vertical stroke represents the trunk
- The horizontal strokes represent branches extending outward
- Together: the basic structure of any tree
- Picture a simple tree silhouette against the sky
- Like the fundamental form that represents all trees

This creates the image: **the archetypal tree structure**.

## Core Meaning & Usage

木 represents **trees, wood, and wooden materials**:

1. **Living trees**: "大树" (dà shù) - "big tree" 
2. **Material**: "木头" (mù tou) - "wood; timber"
3. **Furniture**: "木桌子" - "wooden table"
4. **Construction**: "木房子" - "wooden house"

## Examples

- **树木** (shù mù) - "trees; vegetation"
- **木材** (mù cái) - "lumber; timber"
- **木制** (mù zhì) - "made of wood"
- **原木** (yuán mù) - "raw timber; logs"

## Cultural Context

木 is one of the Five Elements (五行) in Chinese philosophy, representing growth, flexibility, and life force. Trees symbolize longevity, strength, and connection between earth and heaven in Chinese culture.
