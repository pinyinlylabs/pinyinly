In a group; as a unit; together; collectively.

## Quick Reference

| Aspect         | Info                     |
| -------------- | ------------------------ |
| Pinyin         | yī qǐ                    |
| Core meaning   | together; collectively   |
| Part of speech | adverb                   |
| Tone           | 1st + 3rd tone           |

## Visual Breakdown

一起 combines **one + rising together** to show unified action.

| Component | Meaning                  | Contribution to 一起                |
| --------- | ------------------------ | ----------------------------------- |
| **一**    | one; unity; single       | Shows unified action or purpose     |
| **起**    | rise; get up; start      | Represents movement or action together |

## Character Analysis: 起

起 shows **people rising together**:
- **走** (walk/movement) represents coordinated motion
- **己** (self) shows individual participation
- Together: individuals moving as one unit

## Mnemonic

Think of 一起 as **one group rising together**:

- 一 (one) represents unity - everyone moving as a single unit
- 起 (rising) shows coordinated action - like dancers rising in unison
- Picture a group of friends standing up together to leave a restaurant
- Or teammates rising from the bench to celebrate a goal
- The key is synchronized, collective action

## Usage Examples

- **一起吃饭** (yī qǐ chī fàn) - "eat together"
- **我们一起去** (wǒ men yī qǐ qù) - "let's go together"
- **一起工作** (yī qǐ gōng zuò) - "work together"
- **一起学习** (yī qǐ xué xí) - "study together"
- **跟你一起** (gēn nǐ yī qǐ) - "together with you"

## Grammar Patterns

一起 typically:

- **Precedes verbs**: 一起 + verb (collective action)
- **Follows 跟/和**: 跟...一起 (together with...)
- **Can end sentences**: 我们去一起 (let's go together)
- **With pronouns**: 你们一起 (you all together)

## Cultural Context

一起 embodies important Chinese values:

- **Collective harmony**: Emphasis on group unity over individualism
- **Shared experience**: The importance of doing things together
- **Social bonding**: Activities are more meaningful when shared
- **Mutual support**: Working together to achieve common goals
