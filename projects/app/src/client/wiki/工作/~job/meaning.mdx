Activity involving mental or physical effort done to achieve a purpose or result; job; occupation;
task.

## Quick Reference

| Aspect         | Info                                    |
| -------------- | --------------------------------------- |
| Pinyin         | gōngzuò                                 |
| Core meaning   | work; job; occupation; task; employment |
| Part of speech | noun, verb                              |
| Tone           | gōng (1st), zuò (4th)                   |

## Visual Breakdown

工作 combines **craft/work + action** to represent productive activity.

| Component | Visual Description                                        |
| --------- | --------------------------------------------------------- |
| **工**    | Tool/craft - representing skilled labor and construction  |
| **作**    | Action/making - showing the process of doing and creating |

The compound suggests skilled manual work combined with purposeful action and creation.

## Mnemonic

Think of 工作 as **"using tools and skills to create something valuable"**:

- 工 (gōng) represents the craftsperson with tools and skills
- 作 (zuò) shows the active process of making and doing
- Like a master craftsperson actively using tools to build something
- The combination of skill, tools, and productive action
- Creating value through dedicated effort and expertise

This creates the image: **skillful hands actively creating valuable results**.

## Core Meaning & Usage

工作 represents **productive activity, employment, and purposeful effort**. It's used:

1. **Employment**: 找工作 (zhǎo gōngzuò) - "look for a job"
2. **Activity**: 在工作 (zài gōngzuò) - "at work; working"
3. **Profession**: 什么工作 (shénme gōngzuò) - "what job/occupation"
4. **Task**: 完成工作 (wánchéng gōngzuò) - "complete the work"

## Examples

- **上班工作** (shàngbān gōngzuò) - "go to work"
- **工作时间** (gōngzuò shíjiān) - "working hours"
- **工作经验** (gōngzuò jīngyàn) - "work experience"
- **工作环境** (gōngzuò huánjìng) - "work environment"
- **辞职** (cí zhí) - "resign from work"

## Types of Work

- **全职工作** (quánzhí gōngzuò) - "full-time job"
- **兼职工作** (jiānzhí gōngzuò) - "part-time job"
- **临时工作** (línshí gōngzuò) - "temporary work"
- **远程工作** (yuǎnchéng gōngzuò) - "remote work"
- **创业** (chuàngyè) - "start a business/entrepreneurship"

## Work Quality and Performance

- **努力工作** (nǔlì gōngzuò) - "work hard"
- **认真工作** (rènzhēn gōngzuò) - "work conscientiously"
- **高效工作** (gāoxiào gōngzuò) - "work efficiently"
- **工作成果** (gōngzuò chéngguǒ) - "work results"

## Workplace Relations

- **同事** (tóngshì) - "colleague; coworker"
- **老板** (lǎobǎn) - "boss"
- **员工** (yuángōng) - "employee"
- **团队合作** (tuánduì hézuò) - "teamwork"

## Modern Work Concepts

- **工作生活平衡** (gōngzuò shēnghuó pínghéng) - "work-life balance"
- **职业发展** (zhíyè fāzhǎn) - "career development"
- **工作压力** (gōngzuò yālì) - "work pressure/stress"
- **工作满意度** (gōngzuò mǎnyìdù) - "job satisfaction"

## Cultural Significance

工作 in Chinese culture represents:

- **自力更生** (zìlì gēngshēng) - Self-reliance through work
- **劳动光荣** (láodòng guāngróng) - "Labor is glorious"
- **勤劳致富** (qínláo zhìfù) - "Hard work leads to prosperity"
- **敬业精神** (jìngyè jīngshén) - Professional dedication

## Grammar Functions

- **Noun**: 我的工作 (wǒ de gōngzuò) - "my job"
- **Verb**: 在医院工作 (zài yīyuàn gōngzuò) - "work at a hospital"
- **With duration**: 工作了十年 (gōngzuòle shí nián) - "worked for ten years"

## Common Expressions

- **下班** (xià bān) - "get off work"
- **加班** (jiā bān) - "work overtime"
- **请假** (qǐng jià) - "ask for leave"
- **工作狂** (gōngzuò kuáng) - "workaholic"

## Learning Notes

工作 is essential because:

- Fundamental concept for discussing careers and employment
- Essential for adult life and economic participation
- Key to understanding Chinese work culture and values
- Important for business and professional communication
- Demonstrates the connection between skill (工) and action (作)

工作 reflects the Chinese understanding that meaningful work combines skill, effort, and purpose to
create value for society!
