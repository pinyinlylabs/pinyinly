**🗣️ Pronunciation of 面 (miàn)**

- **Pinyin:** miàn
- **Tone: Fourth tone** — sharp **falling** tone, like giving a command

**🔤 Breakdown:**

- **m** like **"m"** in "moon"
- **iàn** sounds like **"yen"** in Japanese yen, but with fourth tone → sharp falling
- **miàn** sounds like **"myen!"** with a sharp drop

**📝 Common Examples:**

- 面 (miàn) - "face; surface"
- 见面 (jiàn miàn) - "meet; see each other"
- 面包 (miàn bāo) - "bread"
- 面条 (miàn tiáo) - "noodles"
- 表面 (biǎo miàn) - "surface; appearance"

**💡 Memory Tip:**

The **sharp falling tone** is like looking directly at someone's face with focus — **miàn**
