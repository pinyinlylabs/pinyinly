**🗣️ Pronunciation of 学 (xué)**

- **Pinyin:** xué
- **Tone: Second tone** — **rising** tone, like asking a question: **"Really?"**

**🔤 Breakdown:**

- **x** like **"sh"** in "sheep"
- **ué** sounds like **"weh"** but with a rising tone
- **xué** sounds like **"shweh?"** with a questioning rise

**🎧 Tone tip:**

The **second tone** (ˊ) is **rising like a question**:

Start **low and rise up** — like enthusiastically asking to learn: **"xué?"** — that's the eager,
rising second tone.

**📝 Common Examples:**

- 学 (xué) - "to learn; to study"
- 学习 (xué xí) - "to study; to learn"
- 学校 (xué xiào) - "school"
- 学生 (xué sheng) - "student"
- 大学 (dà xué) - "university"
- 中学 (zhōng xué) - "middle school"
- 小学 (xiǎo xué) - "elementary school"
- 学会 (xué huì) - "to learn how to; to master"

**💡 Memory Tip:**

The second tone's rising pattern is like eagerly asking **"Can I learn this?"** — that enthusiastic
rise matches the spirit of **learning**!
