**🗣️ Pronunciation of 继 (jì)**

- **Pinyin:** jì
- **Tone: Fourth tone** — sharp **falling** tone, like giving a command

**🔤 Breakdown:**

- **j** like **"j"** in "jeep" (but slightly softer)
- **ì** sounds like **"ee"** in "see", but with fourth tone → sharp falling
- **jì** sounds like **"jee!"** with a sharp drop

**🎧 Tone tip:**

The **fourth tone** (`) is a **sharp falling** tone:

Say it like making a firm decision: **"jì!"** — start high and drop down decisively.

**📝 Common Examples:**

- 继续 (jì xù) - "to continue"
- 继承 (jì chéng) - "to inherit; to carry on"
- 继母 (jì mǔ) - "stepmother"
- 继父 (jì fù) - "stepfather"
- 后继 (hòu jì) - "successor; heir"

**💡 Memory Tip:**

**Continue** with determination — that decisive action matches the sharp falling tone!
