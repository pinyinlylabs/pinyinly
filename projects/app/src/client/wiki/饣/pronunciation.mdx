**🗣️ Pronunciation of 饣 (shí)**

- **Pinyin:** shí
- **Tone: Second tone** — **rising** tone, like asking a question

**🔤 Breakdown:**

- **sh** like **"sh"** in "ship"
- **í** sounds like **"ee"** in "see" but with a rising tone
- **shí** sounds like **"she"** with an upward inflection

**🎧 Tone tip:**

The **second tone** (´) is a **rising** tone. Start lower and rise up, like when you're asking
"Really?" in a questioning voice.

**📝 Note on Usage:**

饣 is the **simplified radical form** of 食 (food). It appears on the left side of many food-related
characters but is rarely used alone. When it does appear independently, it uses the same
pronunciation as 食.

**📝 Common Examples in Characters:**

- 饭 (fàn) - "rice, meal" (contains 饣)
- 饺 (jiǎo) - "dumpling" (contains 饣)
- 饿 (è) - "hungry" (contains 饣)
- 饱 (bǎo) - "full" (contains 饣)
- 饮 (yǐn) - "to drink" (contains 饣)

**💡 Memory Tip:**

This radical appears in almost every food-related character. Remember it shares the same
pronunciation as 食 (shí) with that questioning rising tone.
