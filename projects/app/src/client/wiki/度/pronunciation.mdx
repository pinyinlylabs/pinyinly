**🗣️ Pronunciation of 度 (dù)**

- **Pinyin:** dù
- **Tone: Fourth tone** — sharp **falling** tone, like giving a firm command

**🔤 Breakdown:**

- **d** like **"d"** in "dog"
- **ù** sounds like **"oo"** in "book" but with fourth tone → sharp falling
- **dù** sounds like **"doo!"** with a sharp drop in pitch

**🎧 Tone tip:**

The **fourth tone** (ˋ) is a **sharp falling** tone:

Start high and drop quickly, like when you say "No!" decisively. **dù** should sound firm and
authoritative.

**📝 Common Examples:**

- 度 (dù) - "degree; measure"
- 温度 (wēn dù) - "temperature"
- 态度 (tài dù) - "attitude"
- 角度 (jiǎo dù) - "angle; perspective"
- 速度 (sù dù) - "speed"
- 难度 (nán dù) - "difficulty"

**💡 Memory Tip:**

度 means "degree" — the **sharp downward** fourth tone is like marking a **precise degree** on a
scale with authority!
