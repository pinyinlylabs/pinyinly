The number two; the second whole number; a pair or couple.

## Quick Reference

| Aspect        | Info               |
| ------------- | ------------------ |
| Pinyin        | èr                 |
| Core meaning  | two; pair; second  |
| Part of speech| number, adjective  |
| Tone          | fourth tone (falling) |

## Visual Breakdown

二 consists of **two parallel horizontal strokes**, clearly representing the number two.

| Component | Visual Description |
| --------- | ------------------ |
| **二**    | Two horizontal lines stacked vertically |

## Mnemonic

The character 二 **visually shows two** - imagine two shelves on a wall, two floors of a building, or two lines drawn one above the other.

Think of it as:
- Two horizon lines (maybe a double rainbow!)
- Two rulers stacked on top of each other
- Two chopsticks laid parallel
- The mathematical "equals" sign (=) rotated 90 degrees

## Core Meaning & Usage

二 represents **duality, pairing, and the number two**. It's used:

1. **As a number**: 二个人 (èr gè rén) - "two people"
2. **In compounds**: 二月 (èr yuè) - "February" 
3. **For ordering**: 第二 (dì èr) - "second; number two"
4. **In fractions**: 二分之一 (èr fēn zhī yī) - "one half"

## Examples

- **二十** (èr shí) - "twenty"
- **二手** (èr shǒu) - "second-hand; used"
- **二楼** (èr lóu) - "second floor"
- **星期二** (xīngqī èr) - "Tuesday"

## Learning Notes

二 is essential for:
- All numbers from 12-19, 20-29, etc.
- Ordering and sequencing
- Time expressions (dates, days of week)
- Mathematical concepts

**Note**: In some contexts, especially when counting or giving quantities, Chinese speakers often use 两 (liǎng) instead of 二 for "two." For example, "two books" is usually 两本书 (liǎng běn shū) rather than 二本书. However, 二 is always used in fixed compounds and number sequences.
