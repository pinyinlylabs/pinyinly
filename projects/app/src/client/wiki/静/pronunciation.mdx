**🗣️ Pronunciation of 静 (jìng)**

- **Pinyin:** jìng
- **Tone: Fourth tone** — sharp **falling** tone, like giving a command

**🔤 Breakdown:**

- **j** like **"j"** in "jeep" (but softer, more like "gee")
- **ìng** sounds like **"ing"** in "sing", but with fourth tone → sharp falling
- **jìng** sounds like **"jing!"** with a sharp drop

**📝 Common Examples:**

- 静 (jìng) - "quiet"
- 安静 (ān jìng) - "peaceful; quiet"
- 平静 (píng jìng) - "calm; tranquil"
- 静止 (jìng zhǐ) - "still; motionless"

**💡 Memory Tip:**

The **sharp falling tone** is like a firm "Shh!" command for quiet — **jìng**
